# Don's Dictionary App

A single-page dictionary application built with HTML, CSS, and JavaScript.

## Features

- Search for English words
- View pronunciation
- Listen to pronunciation audio
- View definitions
- View synonyms
- Error handling for invalid words

## API

This project uses the Free Dictionary API.

## Technologies

- HTML
- CSS
- JavaScript
- Fetch API

## How to Run

Open `index.html` in a browser e.g Chrome or use VS Code Live Server.
